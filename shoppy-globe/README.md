# shoppyGlobe-Internshala

ShoppyGlobe is a fully functional e-commerce web application built with React, Vite, Redux Toolkit, and React Router. This project was developed as part of an academic assignment to demonstrate proficiency in modern frontend development practices.

## Features

- **Product Listing**: Fetches and displays a dynamic list of products from the DummyJSON API.
- **Search & Filter**: Real-time product filtering using Redux state management.
- **Cart Management**: Add products, remove items, and adjust quantities (minimum of 1).
- **Dynamic Routing**: Detailed product pages with route parameters.
- **Checkout Flow**: A mock checkout system with order summary and automated redirection.
- **Performance Optimized**: Implements code splitting (React.lazy/Suspense) and lazy loading for images.
- **Responsive Design**: Optimized for mobile, tablet, and desktop screens.
- **Custom 404 Page**: User-friendly error handling for unknown routes.

## Tech Stack

- **Framework**: React (Vite)
- **State Management**: Redux Toolkit & React Redux
- **Routing**: React Router DOM (createBrowserRouter)
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Custom properties & Flexbox/Grid)

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository.
2. Navigate to the project folder: `cd shoppy-globe`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
